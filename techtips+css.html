CSS Selectors

Selector Use Example
Element Selector Apply styling to all given elements img { /* apply style to all img tags */ }
Class Selector Select all elements assigned to a given class
By using the class="" attribute		 .Class { /* applies to all elements with class="Class */ }
Id Selector Select the element with the assigned id
By using the id="" attribute		 #Title { /* will affect only the element with the id of title */ }
Descendant Combiner Select an element that is nested inside another element, class, or id .article p { /* style only <p> tags that are nested inside of elements of class "article" */ }
